#  Редактор бизнес-разборов (Interactive Document Editor)

Этот проект представляет собой интерактивный HTML-редактор, созданный для оформления профессиональных бизнес-анализов и консалтинговых отчетов. Он позволяет быстро редактировать текстовые блоки и сохранять готовый результат в формате PDF с сохранением фирменного стиля.

##  Особенности

* Прямое редактирование: Весь текст на странице можно изменять в реальном времени благодаря атрибуту contenteditable.
* Динамические списки: Удобные кнопки + и − для добавления или удаления пунктов в разделах "Что есть" и "Потеря денег".
* Управление рисками: Специальный блок для работы с зонами риска с автоматической нумерацией (01, 02 и т.д.).
* Оптимизация для печати: Настроенные стили @media print позволяют скачивать документ в формате PDF (A4) через стандартное окно печати браузера.
* Минималистичный дизайн: Использование современных шрифтов *Geologica* и *Playfair Display* для создания эстетики премиального консалтинга.

##  Технологический стек

* HTML5 — семантическая структура документа.
* CSS3 — адаптивная верстка, использование переменных (CSS Variables) и стилей для печати.
* JavaScript (Vanilla JS) — логика добавления/удаления элементов и управления фокусом.
* Google Fonts — типографика.

##  О проекте

Данный инструмент разработан для предпринимателей и консультантов, которым важно быстро превращать сырые данные аудита в структурированный и визуально приятный отчет для клиента.
